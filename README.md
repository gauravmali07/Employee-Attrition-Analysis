# Employee Attrition Analysis Dashboard

## 📌 Project Overview
This project analyzes employee attrition trends and identifies the key factors influencing employee turnover. The dashboard helps HR teams make data-driven decisions to improve employee retention.

## 🎯 Problem Statement
XYZ Company has been experiencing an employee attrition rate of approximately 15% over the past few years. The objective is to analyze employee data, identify the reasons behind attrition, and provide actionable insights.

## 🛠️ Tools & Technologies
- Power BI
- CSV Dataset
- Data Analytics
- HR Analytics

## 📊 Dashboard KPIs
- Total Employees: 4410
- Employee Attrition: 711
- Attrition Rate: 16.12%
- Average Age: 36.92 Years

## 📈 Key Insights
- Attrition by Age Group
- Attrition by Education Field
- Attrition by Job Role
- Attrition by Salary Hike
- Workforce Demographics Analysis
